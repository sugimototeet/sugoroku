# かんたん じんせいゲーム

iPadやスマートフォンのブラウザで遊べる、1ファイルのすごろくゲームです。

## 遊び方

`sugoroku.html` をブラウザで開くと遊べます。

## GitHub Pagesで公開する

1. このフォルダの中身をGitHubリポジトリにアップロードします。
2. GitHubのリポジトリ画面で `Settings` → `Pages` を開きます。
3. `Build and deployment` の `Source` を `Deploy from a branch` にします。
4. `Branch` を `main`、フォルダを `/ (root)` にして保存します。
5. 表示されたURLをiPadのSafariで開きます。

GitHub Pagesでは `index.html` が入口になり、自動で `sugoroku.html` を開きます。
